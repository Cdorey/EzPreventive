### EzNutrition：简易公共卫生医师营养学辅助计算器

尊敬的同行：

在日常公共卫生工作中，我们需要进行食谱计算和营养分析，以优化健康干预措施并提高服务质量。**EzNutrition** 是一个基于 **Blazor WebAssembly** 技术开发的营养学辅助计算工具，能够提供 **本地化营养计算** 和 **膳食建议支持**，帮助专业人员更高效地完成工作。

### **工具特点**

- **本地计算**：除 AI 生成膳食建议外，几乎所有营养学计算均在本地设备上执行，提高隐私性和安全性。
- **开放源码**：遵循 **AGPL-3.0 许可**，用户可查验和调整计算逻辑，确保计算过程的透明度。
- **专业适用性**：
  - 适用于 **注册的公共卫生执业医师** 或 **临床执业医师（预防保健、儿童保健、妇女保健等营养相关专业）**；
  - 适用于 **取得卫生专业技术资格的营养技师** 或 **中国营养学会认证的注册营养师**。
  - 适用于 **公共卫生学院及其他营养相关专业院系的高校师生**。

### **使用限制**

本工具仅限于：

- **专业医护人员** 进行营养计算和分析；
- **公共卫生学院及其他营养相关专业高校院系** 开展教育教学工作时作为教学参考。

**EzNutrition 不构成医学建议**，所有计算结果仅供参考，用户需依据自身专业判断使用，并遵守所在地相关法规。使用过程中，如因不当操作导致损失，EzNutrition 及其开发者概不负责。本程序的计算方法及其所有模型参数基于营养学教科书及营养学会公开发布的参考资料构建，包括但不限于《营养与食品卫生学》、《中国居民膳食指南》、《中国居民膳食营养素参考摄入量2023》和《食物成分表》等权威资料。
